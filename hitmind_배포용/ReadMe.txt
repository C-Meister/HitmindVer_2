HitMind 프로젝트
팀명 C Meister team - 서상희 배수한 신상호 장민석
dll파일이 숨겨져 있으니 폴더를 통째로 복사를 해 주시길 바랍니다.
게임내 명령어는 
전체 채팅
- /clear - 채팅 지우기

- /refrash - 화면 재출력 (화면이 보이지 않을때 사용 바람)

[관리자용]
- /mysql [쿼리] 서버 MySQL과 질의응답을 수행함.
성공시 [MySQL] Query Success
실패시 [MySQL] Query Error 출력

게임 내 채팅
- /screenshot [파일명] 현재 그리고 있는 화면을 캡쳐하여 screenshot폴더에 파일명으로 저장이 됩니다.

※사운드 파일이나 이미지파일을 손상할 시 제대로 실행이 안될 수 있습니다.
※프로필 사진을 바꾸는 것은 MySQL 질의응답으로만 실행이 됩니다. (UI구현은 안되어 있습니다.)
